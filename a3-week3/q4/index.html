Superscript

Superscript tag <sup> as it says renders the text inside the tag raised above the normal baseline with smaller text. This tag can be used to represent mathematical equations like powers.

Ex: a2 + b2 + 2ab = (a+b)2

  <p>a<sup>2</sup> + b<sup>2</sup> + 2ab = (a+b)<sup>2</sup></p>

Subscript

Subscript tag <sub> renders the text inside the tag raised below the normal baseline with smaller text. This tag can be used to represent mathematical variables or Chemical formulas.

Ex: H2O

  <p>H<sub>2</sub>O</p>

Strike Through

The Strike Through tag <strike> is used to strike out the content inside it. It can be used for applications like todo app to strike out the task after completition to act as a good visual guide that the task has been completed

Ex: This text is striked

  <p><strike>This text is striked</strike></p>
